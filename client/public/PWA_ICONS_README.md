# PWA Icons - Instrucciones

Para tener una PWA completa, necesitas crear estos archivos en `client/public/`:

## Iconos requeridos:

1. **pwa-192x192.png** (192x192 px)
   - Icono pequeño para lista de apps

2. **pwa-512x512.png** (512x512 px)
   - Icono grande para splash screen
   - También usado como maskable icon

3. **apple-touch-icon.png** (180x180 px)
   - Icono para iOS cuando se agrega a pantalla de inicio

4. **mask-icon.svg** (SVG monocromo)
   - Icono para Safari pinned tabs

5. **favicon.png** (Ya existe)
   - Icono en pestaña del navegador

## Generadores online recomendados:

- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator
- https://maskable.app/ (para maskable icons)

## Diseño sugerido:

- Fondo: #0ea5e9 (azul sky-500)
- Logo: FacturaXpress o iniciales "FX"
- Estilo: Moderno, profesional, clean
- Forma: Cuadrado con bordes redondeados

## Ubicación:
Todos los iconos deben estar en: `client/public/`
